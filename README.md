# counting-words

Цей скрипт призначений для ефективного підрахунку кількості вживань кожного слова у текстовому файлі. Використання **Streams API** дозволяє обробляти навіть дуже великі файли без ризику переповнення оперативної пам'яті.

## Функціонал
* **Стрімінгове читання:** Файл обробляється частинами (chunks), що забезпечує стабільну роботу з великими даними.
* **Автоматичне сортування:** Результати виводяться у вигляді таблиці, відсортованої від найбільш вживаних слів до найменш вживаних.
* **Мінімальні залежності:** Використовується тільки вбудований модуль `node:fs`.

## Вимоги
* Встановлений [Node.js](https://nodejs.org/) (версія 16 або вище).

## Як запустити

1.  **Скопіюйте код** у файл, наприклад `index.js`.
2.  **Підготуйте текстовий файл** (наприклад, `data.txt`), який ви хочете проаналізувати.
3.  **Виконайте команду** у терміналі, передавши шлях до файлу як аргумент:

```bash
node index.js data.txt
```

##  Приклад виводу
Після завершення обробки ви побачите таблицю в консолі:

| (index) | 0 | 1 |
| :--- | :--- | :--- |
| 0 | 'привіт' | 42 |
| 1 | 'код' | 15 |
| 2 | 'Node.js' | 8 |

*Де `0` — це слово, а `1` — кількість його повторів.*

##  Особливості реалізації
На даний момент скрипт:
1.  **Чутливий до регістру:** слова "Привіт" і "привіт" будуть рахуватися як різні.
2.  **Враховує пунктуацію:** якщо після слова стоїть кома (наприклад, `привіт,`), вона буде вважатися частиною слова.
3.  **Економний:** використовує константний обсяг пам'яті незалежно від розміру файлу (крім об'єкта зі списком унікальних слів).
